# DÖNERRECHNER
## WAS IST DER DÖNERRECHNER?
Viele Menschen stellen sich oft die Frage "Mh.. Ist es das Geld wert, was ich dafür ausgeben würde?

Die Antwort ist meistens Nein.  
Der Dönerrechner bietet dir Möglichkeit, alle Preise in Döner umzurechner, so kannst du besser entscheiden, ob du das Geld dafür ausgeben möchtest

## Ich könnte doch auch einfach den Rechner nutzen
Ja.. Nein!  
Unsere Web-App und API Schnittstelle sind der Weg in eine bessere Döner-Berechnung. Wir merken uns, wie teuer dein Döner das letzte mal war und bieten dir in Zukunft bestimmt coole Sachen, wie eine Browser-Erweiterung!
